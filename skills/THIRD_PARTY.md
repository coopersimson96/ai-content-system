# Third-Party Skills

The following skills are authored by external creators and republished here with attribution under their original open-source licenses. Updates flow upstream first.

## Hyperframes Suite — Nate Herkai / Heygen

Six skills pulled from [github.com/heygen-com/hyperframes](https://github.com/heygen-com/hyperframes), licensed Apache 2.0:

- `hyperframes/` — HTML composition authoring, GSAP timelines, audio sync
- `hyperframes-cli/` — CLI commands (init, lint, preview, render, transcribe, tts, doctor)
- `hyperframes-registry/` — install and wire registry blocks/components
- `gsap/` — animation reference for HyperFrames compositions
- `website-to-hyperframes/` — capture a website and convert to Hyperframes video
- `claude-design-hyperframes/` — Claude Design wrapper for Hyperframes drafts

To pull the latest upstream versions independently of this repo:
```bash
npx skills add heygen-com/hyperframes
```

The `video-editor/` skill in this repo is a workflow wrapper around these. It documents the 9:16 short-form pipeline, three style archetypes (sage-stack, editorial-magazine, screen-face-split), and the orange frosted-glass + karaoke pill overlay defaults.
